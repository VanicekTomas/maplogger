CartoLogger Extension

This tab extends the original MapLogger idea with support for CartoLogger CSV exports. CartoLogger is a Chrome extension that records map interactions (clicks, pans, zooms, and URL modifications) and exports logs for behavioural analysis.

The analysis here focuses on URL-based interaction traces, especially url_change events, and explains what changed between consecutive URLs (zoom, map layer, coordinates, and related parameters).

Tool source: CartoLogger GitHub repository. Example study use: ICCGIS 2024 paper (PDF).
